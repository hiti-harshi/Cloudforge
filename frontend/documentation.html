🚀 Launching a Lab

1. Open Dashboard.
2. Select Ubuntu or Linux Lab.
3. Click Launch Lab.
4. Wait for the environment to be provisioned.

🔒 Connecting via SSH

ssh -i your-key.pem ubuntu@PUBLIC_IP

Replace PUBLIC_IP with the IP displayed after launching your lab.
